Para utilizar o programa va, atraves do terminal, ate o diretorio com os arquivos
de codigo fonte Arvore.h, Cadastro.h, main.cpp, PacienteComEmail.cpp, Paciente.cpp,
PacienteComIdade.cpp, Paciente.h, PacienteComEmail.h, PacienteComIdade.h e makefile.
Em seguida digite no terminal "make" para gerar o executavel do programa. Entao,
digite "./cadastro" para executar e selecione entre as opcoes oferecidas pelo menu do
programa. Pode-se utilizar o comando "make clean" no terminal para deletar o executavel.
